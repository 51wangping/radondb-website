RadonDB MySQL Kubernetes 于 3 月 24 日发布版本 2.1.3。该版本主要基于在 2.1.2 进行功能优化和升级。

# **致谢**

首先感谢 @andyli029 @acekingke @runkecheng @mgw2168 @molliezhang 提交的修改。


# **新版本功能一览**

1. 一键发布工作流
2. 支持按标签重建集群节点
3. 增加 Pod 调试模式

以下是完整的 Release Notes。
